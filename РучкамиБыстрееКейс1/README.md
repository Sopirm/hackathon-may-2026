# Робот разнесения банковской выписки в 1С (Кейс 1)

## Описание
Робот читает банковскую выписку (Excel, CSV или JSON), для каждой строки определяет поступление или списание, создаёт новую выписку, находя контрагента в 1С по ИНН и сохраняет отчёт о результатах.

## Требования
- Puzzle RPA Studio 3.0.3.
- Доступ к HTTP-сервису 1С (BASE_URL, LOGIN, PASSWORD – выданы организаторами)

## Установка и запуск
1. Распакуйте архив с проектом.
2. Откройте проект в Puzzle RPA Studio.
3. Заполните конфиг в папке `resource/`.
   Пример конфига
```
{
  "base_url": "https://demo.mirea.ru/1c_api/",
  "login": "demo_user",
  "password": "demo_password",

  "statement_file": "statement.xlsx",
  "report_file": "report.csv"
}
```
5. Поместите файл выписки `statement.xlsx` в папку `resource/` (согласно примеру формата).
6. Запустите робота: откройте `process/main` и нажмите F5.

## Формат выписки (Excel)
| Дата | Сумма | Тип | Назначение |   Организация   |
|------|-------|-----|------------|-----------------|

## Результат
Файл `resource/report.csv` с колонками: Дата, Сумма, Контрагент, Статус, Сообщение.
