# Робот разнесения банковской выписки в 1С (Кейс 1)

## Описание
Робот читает банковскую выписку (Excel), для каждой строки определяет поступление или списание, находит контрагента в 1С по ИНН, создаёт соответствующий документ и сохраняет отчёт о результатах.

## Требования
- Puzzle RPA Studio 3.0.2.
- Доступ к HTTP-сервису 1С (BASE_URL, LOGIN, PASSWORD – выданы организаторами)

## Установка и запуск
1. Распакуйте архив с проектом.
2. Откройте проект в Puzzle RPA Studio.
3. В блоках «Отправить запрос в 1С» укажите выданные BASE_URL, LOGIN, PASSWORD.
4. Поместите файл выписки `statement.xlsx` в папку `resource/` (согласно примеру формата).
5. Запустите робота: откройте `process/main.robot` и нажмите F5.

## Формат выписки (Excel)
| Дата | Сумма | Тип | Назначение | ИНН_контрагента |
|------|-------|-----|------------|-----------------|

## Результат
Файл `resource/report.csv` с колонками: Дата, Сумма, Контрагент, Статус, Сообщение.
