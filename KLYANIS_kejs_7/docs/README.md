# Документация

## Краткое описание решения

Мы выбрали кейс 7: разработка пользовательского блока для Puzzle RPA, который форматирует Word-документы по заданным правилам. Блок должен принимать путь к `.docx` файлу, опциональный путь сохранения, параметры шрифта и словарь замен для плейсхолдеров вида `{{ключ}}`.

Архитектура решения строится как расширение Puzzle RPA: описание блока находится в `block.json`, размещение в тулбоксе задается через `values.xml`, вызов Python-функции выполняется через `code.py`, а основная логика размещается в `src/__init__.py`. Планируется использовать пользовательский блок `word_formatter`, который будет работать с Microsoft Word через COM-объекты `win32com`.

## Текущий статус

Уже реализован начальный каркас расширения `word_formatter` в папке `solution`: добавлены служебные файлы Puzzle RPA, настроено отображение блока в категории `Интеграция приложений / Word`, создана точка входа `format_document` и добавлена базовая валидация параметров.

Сейчас в работе находится основная логика форматирования Word-документа: открытие файла через COM, очистка лишнего форматирования, применение единого шрифта, определение заголовков, оформление таблиц и замена плейсхолдеров.
