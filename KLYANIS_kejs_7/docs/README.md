## Кейс

Выбран кейс 7: разработка пользовательского блока для Puzzle RPA Studio, который автоматически форматирует Word-документы `.docx`. Блок приводит документ к единому стилю, заменяет плейсхолдеры, определяет заголовки и оформляет таблицы.

## Описание решения

Решение реализовано в виде пользовательского расширения `word_formatter` для Puzzle RPA. Расширение подключается в студию как отдельный блок и использует автоматизацию Microsoft Word через COM-интерфейс `win32com.client`. Такой подход соответствует требованиям кейса и позволяет работать с настоящим Word-документом, а не с его упрощенной текстовой копией.

Архитектурно блок состоит из описания интерфейса для Puzzle RPA и Python-логики форматирования. В текущей версии основная рабочая функция находится в `solution/word_formatter/code.py`. Блок принимает путь к исходному файлу, путь сохранения результата, параметры шрифта и словарь замен, после чего запускает Word в фоновом режиме, обрабатывает документ и сохраняет итоговый `.docx`.

## Что делает блок

Блок `word_formatter` выполняет следующие действия:

- проверяет входные параметры и существование файла;
- проверяет, не открыт ли исходный или выходной файл в Microsoft Word;
- заменяет плейсхолдеры вида `{{ключ}}` на значения из словаря;
- очищает документ от табуляций, лишних пробелов и части пустых абзацев;
- задает единый шрифт и размер шрифта всему документу;
- автоматически определяет заголовки первого и второго уровня;
- форматирует таблицы: включает границы, выравнивает текст, выделяет первую строку как заголовочную;
- сохраняет результат в исходный файл или в новый путь.

## Параметры блока

| Параметр | Тип | Обязательный | По умолчанию | Назначение |
|----------|-----|--------------|--------------|------------|
| `path_file` | string | Да | нет | Путь к исходному `.docx` файлу |
| `path_file_end` | string | Нет | исходный файл | Путь для сохранения результата |
| `font_name` | string | Нет | `Calibri` | Основной шрифт документа |
| `font_size` | int | Нет | `12` | Размер шрифта |
| `replacements` | dict | Нет | `{}` | Словарь замен для плейсхолдеров |

## Требования к окружению

- Windows 10 или Windows 11
- Microsoft Word 2016 или новее
- Puzzle RPA Studio 3.0.3

## Установка расширения

1. Откройте Puzzle RPA Studio.
2. Перейдите в раздел `Настройки -> Расширения -> Добавление`.
3. Укажите папку [solution/word_formatter](/Users/sopirm/ProgramProjects/hacatonRPA/hackathon-may-2026/KLYANIS_kejs_7/solution/word_formatter).
4. Подтвердите установку расширения.
5. Перезапустите Puzzle RPA Studio.
6. После перезапуска блок появится в категории `Интеграция приложений / Word`.

## Инструкция запуска

1. Подготовьте входной файл [test_input.docx](/Users/sopirm/ProgramProjects/hacatonRPA/hackathon-may-2026/KLYANIS_kejs_7/test_input.docx).
2. В Puzzle RPA перетащите блок `Форматировать Word документ` на рабочую область.
3. Заполните параметры:
   `path_file`: абсолютный путь к `test_input.docx`
   `path_file_end`: абсолютный путь к новому файлу результата, например `C:\\test\\result.docx`
   `font_name`: `Calibri`
   `font_size`: `12`
   `replacements`: `{"name": "Иван", "company": "ООО Тест"}`
4. Запустите процесс.
5. На выходе получите отформатированный документ. Для сравнения можно использовать [test_output.docx](/Users/sopirm/ProgramProjects/hacatonRPA/hackathon-may-2026/KLYANIS_kejs_7/test_output.docx).
